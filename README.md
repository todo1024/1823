课程介绍：

本课程将带领大家学习HDFS架构、数据存储策略、企业级分布式集群搭建、数据存储、读取原理、高可用、HDFS 命令及API企业级实战应用等技能点，如果你正在学习大数据或者在工作中使用到了分布式数据存储相关技术，本课程一定适用与你，学习本科课程不仅可以在底层原理上对数据存储有深度认识还可以学习到企业实际开发中如何使用Hadoop HDFS ,扫除你学习工作过程中的各种问题，为大数据学习进阶铺平道路。
